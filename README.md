# ArcShot
# 🎮 2D Projectile Platformer

A fast-paced 2D platformer that combines precision movement with projectile-based combat. Players navigate through challenging levels, using skillful jumping and strategic shooting to defeat enemies and overcome obstacles.

---

## 🚀 Features

- **Classic 2D Platforming Mechanics**  
  Smooth movement system including jumping, gravity, and collision handling.

- **Projectile-Based Combat**  
  Players can shoot projectiles to eliminate enemies and interact with the environment.

- **Enemy Encounters**  
  Dynamic obstacles and enemies that require timing and accuracy to defeat.

- **Level Design**  
  Multiple levels designed to challenge player reflexes and decision-making.

- **Responsive Controls**  
  Designed for tight, responsive gameplay to enhance player experience.

---

## 🛠️ Technologies Used

- Language: C#
- Engine/Framework: Unity
- Other Tools: Visual Studio, Git

---

## 🎮 How to Play

- Use movement controls to navigate platforms  
- Jump to avoid obstacles and reach higher areas  
- Shoot projectiles to defeat enemies  
- Progress through levels by reaching the goal
